# JuJiuBot Gen2 - 菊酒牛牛
我是来自米诺斯的祭司帕拉斯，会在罗德岛休息一段时间......

虽然这么说，我渴望以美酒和戏剧被招待，更渴望走向战场。

## 警告
该项目仍处于 **极早期开发状态**。

## 已开发的功能
- [x] 牛牛聊天
- [ ] 牛牛人品
- [ ] 牛牛占卜/牛牛塔罗牌
- [ ] 牛牛唱歌
- [ ] 牛牛画图
- [ ] 牛牛闭嘴
- [ ] 牛牛今天吃/喝什么
- [ ] 牛牛对联
- [ ] 牛牛喝酒/酒后对话
- [ ] 牛牛夺舍
