## 警告
该项目仍处于 **极早期开发状态**。

## 已开发的功能
- [ ] 牛牛聊天
- [ ] 牛牛人品
- [ ] 牛牛占卜/牛牛塔罗牌
- [ ] 牛牛唱歌
- [ ] 牛牛画图
- [ ] 牛牛闭嘴
